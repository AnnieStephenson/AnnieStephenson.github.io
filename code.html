The projects below are sampling of projects I’ve developed in small teams, with the hope that they can be useful to others. These packages are either actively maintained or could be brought back to active maintenance subject to community interest. If you have a suggestion or encounter a bug, please submit an issue on github or email me, and I’d be happy to work with you to get it implemented or fixed.

Though I’ve taken various coding and data analysis courses, most of what I’ve learned has been through an informal research setting. Blogs and online course hubs have therefore been very helpful resources. Most recently, I’ve been working through Andrew Ng's 5-course Deep Learning Specialization on Coursera.

r-place-emergence

Python package for exploring emergent collective phenomena in the r/place datasets

structural-color

Python package for modeling structural color in colloidal systems.

python-mie

Pure Python package for Mie scattering calculations.

mstm-wrapper

This package provides a python wrapper around the MSTM fortran-90 code written by Daniel Mackowski.

ColorPy

This is a fork from Mark Kness' ColorPy, just updated for Python 3 and with a module for converting reflectance spectra into colors.
